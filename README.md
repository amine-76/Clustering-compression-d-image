# Clustering Compression d'Image

Clustering Compression d'Image est un projet visant à explorer l'utilisation des techniques de clustering, telles que K-Means, pour compresser et réduire la taille des images. Grâce à ces techniques, il est possible de convertir une image à haute définition en une image plus légère tout en préservant une grande partie de ses détails visuels.

## Fonctionnalités

- Implémentation des algorithmes de clustering, comme K-Means, appliqués 
aux données d'image.
- Réduction des couleurs d'une image à l'aide de la quantification
- Conversion effective d'une image pour différentes tailles de clusters.
- Visualisation des résultats avant et après compression.

## Techniques et technologies utilisées

- **Python** et **Jupyter Notebook** pour une analyse interactive et des visualisations.
- Numpy, Pandas, et d'autres bibliothèques pour le traitement de données.
- Bibliothèques de manipulation et de visualisation d'image telles que Pillow et Matplotlib.

## Structure du projet

- `notebooks/` : Contient les notebooks interactifs illustrant le fonctionnement du projet.
- `examples/` : Images exemples utilisées pour tester les modèles de clustering.
- `outputs/` : Résultats de compression générés.

## Comment démarrer ?

1. Clonez ce dépôt :

```
git clone https://github.com/amine-76/Clustering-compression-d-image.git
```

2. Accédez au dossier du projet:

```
cd Clustering-compression-d-image
```

3. Assurez-vous d'avoir installé Python et Jupyter. Installez également les dépendances du fichier `requirements.txt` :

```
pip install -r requirements.txt
```

4. Lancez l'un des notebooks disponibles pour explorer les fonctionnalités du projet.
